## Funcion Cliente Aleatorio

- Debe devolver un DNI
- Usar los parametros de LIMIT cantidad,cuantos
- Ejemplo Limit 5,1 -- trae el sexto dato o id=6
- Si el registro no existe devuelve un **null**
- Por lo que para poder barrer una tabla completa debo restar 1 al dato aleatorio

```sql
    USE `credicel_empresa`;
    DROP function IF EXISTS `f_cliente_aleatorio`;
    DELIMITER $$
        USE `credicel_empresa`$$
        CREATE FUNCTION `f_cliente_aleatorio` ()
        RETURNS VARCHAR(20)
        BEGIN
            DECLARE vresultado VARCHAR(20);
            DECLARE vmax INT;
            DECLARE valeatorio INT;
            SELECT COUNT(*) INTO vmax FROM clientes;
            -- buscamos el valor aletorio entre 1 y maximo de la tabla
            SET valeatorio=f_aleatorio(1,vmax)-1;
            -- buscando el dni
            SELECT 	DNI INTO vresultado 
                    FROM clientes
                    LIMIT valeatorio,1;
        RETURN vresultado;
        END$$
    DELIMITER ;
```
