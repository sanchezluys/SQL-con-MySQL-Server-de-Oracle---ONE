### Usando el SELECT

- SELECT * FROM tabla
- SELECT DNI, NOMBRE FROM tabla
- SELECT DNI AS CEDULA, NOMBRE AS NOMBRE FROM tabla
- con WHERE
- con >, <

### Consultas Condicionales

Se usan tablas de la verdad, logica binaria

![Tabla de la Verdad](/imagenes/tabla_verdad.png)

- OR
- AND
- NOR (NOT OR)
- NAND (NOT AND)
  
### Aplicando Filtros WHERE

```sql
  SELECT 
        * 
  FROM
      PRODUCTOS
  WHERE
      SABOR ='Mango'
      AND TAMAﾃ前='470 ml'
```

```sql
  SELECT 
        * 
  FROM
      PRODUCTOS
  WHERE
      SABOR ='Mango'
      OR TAMAﾃ前='470 ml'
```

```sql
  SELECT 
        * 
  FROM
      PRODUCTOS
  WHERE
      NOT (SABOR ='Mango')
      AND TAMAﾃ前='470 ml'
```

### Usando el WHERE IN()

-Es equivalente a OR anidados

```sql
  SELECT 
        * 
  FROM
      PRODUCTOS
  WHERE
      SABOR IN ('Mango','Uva')
      AND TAMAﾃ前='470 ml'
```

### Usando filtros con LIKE

```sql
  SELECT 
        * 
  FROM
      PRODUCTOS
  WHERE
      campo LIKE '%condicio'
```

- '%suares%' : todos los apellidos suarez
- '%__a%' : la tercera letra es a
